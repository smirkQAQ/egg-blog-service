## 主要技术栈 
```
nodejs, koa2, mysql
```
### 启动
```
npm install
```

* 开发环境启动     
   
```
npm run dev
```
      
* 生产环境启动    
   
```
npm run start
```

#### 项目结构

* 入口文件：  app.js   

* 项目配置文件:    config/*

* 工具库 :  utils/*     
  
* 接口router配置 :  routes/*     
      
* schema/数据库 :  models/*      
  
* 控制器 :  controllers/*     

* 公共中间件 :  middlewares/*
* 
* 接口逻辑 :  service/*
